# CodeTalk Blog

## Installation

To set up CodeTalk Blog on your local machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/jdgiancola/codetalk.git

2. Navigate to the project directory:
`cd codetalk`

3. Install dependencies:
   `npm install`

4. Start the development server:
   `npm start`

5. Open your web browser and visit http://localhost:3000 to view the blog.

## Usage

You can now start writing and publishing your blog posts. Simply add your Markdown files to the posts directory and follow these naming conventions:

Blog post filenames should be in the format YYYY-MM-DD-title.md.
Replace YYYY-MM-DD with the publication date of your post.
Replace title with a descriptive title for your post.
